ИНСТРУКЦИЯ: Как запустить Telegram-бота "Итальянский брейнрот"

1. Установи Python (если ещё не установлен) — https://www.python.org/downloads/

2. Открой терминал или командную строку и перейди в папку, куда распаковал архив.

3. Установи нужные библиотеки:
   pip install -r requirements.txt

4. Запусти бота:
   python main.py

Бот будет работать и отвечать тебе в Telegram. Если не отвечает — убедись, что токен бота из BotFather правильный и у тебя есть интернет.
